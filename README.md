# Jinn

Jinn is a collection of NLP related packages for the Go ecosystem. Jinn is not NLTK. 

## Packages

| Package      | Description                                    |
|--------------|------------------------------------------------|
| `corenlp`    | Simple CoreNLP wrapper                         |
| `msrpc`      | Microsoft Research paraphrase corpus utilities |
| `tree`       | Basic tree implementation                      |
